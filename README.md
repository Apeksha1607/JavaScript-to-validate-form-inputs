# ✅ Registration Form with Validation

This project is a simple **HTML and JavaScript Registration Form** that performs **basic client-side form validation** before submitting the data.

It checks whether required fields are filled and ensures the email address follows a valid format.

---

## 📌 Features

* Simple **registration form**
* Input fields for:

  * Name
  * Email
  * Gender
  * Date of Birth
* **JavaScript form validation**
* Alerts users when:

  * Name field is empty
  * Email field is empty
  * Email format is invalid
* Success message when the form is valid

---

## 🛠 Technologies Used

* **HTML5**
* **JavaScript (Client-side validation)**
